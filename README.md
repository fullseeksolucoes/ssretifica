# SS Retífica — Site Institucional

Site institucional da SS Retífica, especializada em retífica de motores, usinagem de componentes e recondicionamento de motores nacionais e importados.

## Tecnologias

- Next.js
- TypeScript
- Tailwind CSS v4
- shadcn/ui

## Desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) para visualizar.
